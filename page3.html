Network Devices

Device Function Key Features
Repeater Extends network signal range Regenerates weak signals to maintain signal strength. 2-port device.
Hub Connects multiple devices; basic signal repeater Does not filter data; sends data packets to all connected devices.
Bridge Connects two LANs; filters data by MAC addresses Operates at data link layer; transparent or source routing types available.
Switch Connects multiple devices; selectively forwards data Operates at data link layer; divides collision domain; various types including unmanaged, managed, layer 2, layer 3, etc.
Router Routes data based on IP addresses Connects LANs and WANs; dynamically updates routing table; divides broadcast domains.
Gateway Connects networks with different protocols/models Acts as a protocol converter; can operate at any network layer; more complex than switches or routers.
Brouter Combines features of bridge and router Can work at data link or network layer; capable of routing packets and filtering LAN traffic.
NIC Network adapter for connecting computer to network Installs in computer to establish LAN connection; operates at physical and data link layers.
